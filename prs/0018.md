# fix: configure reqwest TLS provider

**PR #18** | CLOSED | Author: charliecreates[bot] | Created: 2026-04-01 | Merged: 2026-04-01

Fixes the `reqwest` configuration to use a concrete TLS provider (via the `rustls` feature) instead of `rustls-no-provider`, so native HTTPS calls work out of the box.

This is a follow-up from the PR #17 review thread.
