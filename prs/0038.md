# fix(settlement): tighten ISO 20022 XML validation

**PR #38** | CLOSED | Author: charliecreates[bot] | Created: 2026-04-16 | Merged: 2026-04-18

Branch: `charlie/iso20022-pre-root-guard` → `master`

Follow-up hardening for the ISO 20022 trigger parser merged in #36.

## Description
- Centralize `xmlns` / `xmlns:*` parsing into `build_namespace_scope` so `Start` and `Empty` branches cannot drift.
- Tighten ISO 20022 namespace detection to require the base URN or the base URN + `:` boundary (rejects lookalike namespaces).
- Reject non-text XML events before the `<Document>` root (only XML declaration + whitespace are tolerated).
