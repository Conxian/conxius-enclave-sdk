## Security boundary
The common boundary binds the exact 32-byte operation digest, signing purpose, algorithm, derivation/key identity, expected returned public key, attestation nonce, canonical signed report, freshness/future skew, provider/root/chain policy, and cryptographic operation signature. Default/WASM construction uses `UnavailableEnclave`; software reports remain typed as software/simulated and cannot satisfy the production policy.
