**PR #31** | CLOSED | Author: charliecreates[bot] | Created: 2026-04-16 | Merged: 2026-04-16

Follow-up fixes from the PR #27 review threads (CON-409).

## Description
- `CloudEnclave`: removes the fixed fallback key, makes `with_dev_key` fallible, and stores dev key material in `Zeroizing<[u8; 32]>`.
- `CoreEnclaveManager::unlock`: derives the session key directly into a `Zeroizing` buffer (avoids extra stack copies).
- `SettlementManager::verify_trigger` (ISO 20022): replaces substring checks with lightweight streaming XML validation via `quick-xml`.
