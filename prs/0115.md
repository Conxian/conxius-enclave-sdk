<blockquote>
<h2>v3.0.0</h2>
<h2>What's changed</h2>
<p>gitleaks-action v3 migrates the runtime from Node 20 to Node 24. <strong>No changes to inputs, outputs, or behavior.</strong> Update your workflow from <code>gitleaks/gitleaks-action@v2</code> to <code>gitleaks/gitleaks-action@v3</code>.</p>
<h3>Migration</h3>
<pre lang="yaml"><code># Before
- uses: gitleaks/gitleaks-action@v2
<h1>After</h1>
<ul>
<li>uses: gitleaks/gitleaks-action@v3<br />
</code></pre></li>
</ul>
<h3>Why</h3>
<p>GitHub is deprecating the Node 20 runtime for Actions:</p>
<ul>
<li><strong>June 2, 2026:</strong> GitHub flips the runner default to Node 24. Workflows using <code>gitleaks-action@v2</code> (Node 20) will still run, but only if <code>ACTIONS_ALLOW_USE_UNSECURE_NODE_VERSION=true</code> is set as an environment variable.</li>
<li><strong>September 16, 2026:</strong> Node 20 is removed from GitHub-hosted runners entirely. <code>gitleaks-action@v2</code> stops working regardless of any opt-out flag.</li>
</ul>
<h3>Changes</h3>
<ul>
<li><code>action.yml</code>: runtime <code>node20</code> → <code>node24</code></li>
<li><code>@actions/core</code>: 1.10.0 → 1.11.1</li>
<li><code>dist/</code> rebuilt</li>
<li>Example workflows updated to <code>actions/checkout@v6</code> and <code>gitleaks-action@v3</code></li>
<li>README updated with v3 migration guide</li>
</ul>
<h3>Self-hosted runners</h3>
<p>If you use self-hosted runners, ensure your runner version is <code>&gt;= v2.327.1</code> (required for Node 24 support).</p>
