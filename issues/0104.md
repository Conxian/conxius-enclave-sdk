# Normalize default branch to main and align branch policy

**Issue #104** | CLOSED | Created: 2026-06-08 | Updated: 2026-06-11

The broader Conxian public portfolio mainly uses `main`, while this repository still uses `master` as the default branch.

Requested follow-up:

* evaluate renaming the default branch from `master` to `main`
* update branch protections, contributor documentation, and automation references accordingly
* keep the change synchronized with any release or documentation workflow updates

Positioning check before merge:

* docs and automation changes should preserve the rule that Conxian is a non-custodial control/orchestration layer
* branch and docs updates must not introduce wording that implies customer-fund custody or customer-data system-of-record ownership

Outcome:

* contributor guidance and org-level automation assumptions become more consistent across the public portfolio
